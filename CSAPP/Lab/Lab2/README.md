# Secret Phase

爆炸次数：0

评价：很显然有隐藏关，进入的方法就是在Phase3后面加上一个字符串`DrEvil`。本题考察递归，最终要使`eax=2`，
通过画图发现有两个分支`eax=2*eax+1`和`eax=2*eax`，`eax`会在前者初始化为0，因此依次调用前者和后者就能让
`eax`变成2，而达成的条件可以参考内存
然后查看内存可解，比较繁琐。
